# 📊 Sistema de Cálculo de Frequência Escolar

Sistema simples e eficiente para cálculo de frequência escolar, desenvolvido para auxiliar no preenchimento de formulários de acompanhamento.

---

## 🎯 Objetivo

Facilitar o cálculo da frequência dos alunos, evitando erros manuais e agilizando o trabalho da secretaria escolar.

---

## 🏫 Aplicação

Projeto desenvolvido para uso na:

**Escola José Martinho Cerpa de Menezes**

---

## ⚙️ Funcionalidades

✅ Cálculo automático para alunos até a 4ª série  
✅ Cálculo automático para alunos a partir da 5ª série  
✅ Resultado em porcentagem (%)  
✅ Destaque visual:
- 🟢 Frequência ≥ 75%  
- 🔴 Frequência < 75%  

✅ Interface simples e intuitiva  
✅ Funciona offline (não precisa de internet após abrir)  

---

## 📐 Fórmulas Utilizadas

### 📘 Até a 4ª série:
Frequência = (Dias frequentados × 100) ÷ Total de dias letivos

### 📗 A partir da 5ª série:
Frequência = (Aulas frequentadas × 100) ÷ Total de aulas do mês
